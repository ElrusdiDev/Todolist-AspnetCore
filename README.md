# Todolist-AspnetCore
ini adalah project aplikasi untuk mencatat daftar tugas  atau to do list.
menggunakan AspNet Core

pada aplikasi ini terdapat proses:

create new to do list
- task name
- task category 
- tanggal mulai
- deadline

Update progress status todo
- Next to do
- Inprogress
- Done
- tanggal update todo

Report
- Todo completed
- Todo Not completed
